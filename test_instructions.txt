
Follow these instructions to test this project in local your setup,,

1. Before Running the project in local, Do Comment the 'manifest.json'-linking code in `frontend/app/layout.js` in <head> tag
2. Then before committing the changes made, Remember to Un-Comment 'manifest.json'-linking code in `layout.js`
3. Also Before Committing to GitHub, Make sure to run these at `repo-root` { ./coflow } -->>
        ```    npm run format:check
               npm run format:fix
               npm run lint:check
               npm run lint:fix    ```
